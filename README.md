# 🕷️ Spider-Verse Fanpage

> "Com grandes poderes vêm grandes responsabilidades..." — e um CSS bem estruturado!

Este é um projeto de Front-End focado na criação de uma fanpage dedicada à trajetória do Homem-Aranha nos cinemas. A página reúne informações, curiosidades e o visual marcante das três eras dos heróis nas telonas (Tobey Maguire, Andrew Garfield e Tom Holland).

## 🚀 Funcionalidades da Página

* **Linha do Tempo dos Filmes:** Seção organizada mostrando a evolução dos filmes de 2002 até os mais recentes.
* **Galeria dos Miranhas:** Cards estilizados destacando os três atores principais e seus respectivos trajes.
* **Design Temático:** Uso de paleta de cores inspirada nos quadrinhos (Vermelho, Azul e Dark Mode estilizado).
* **Responsividade:** Layout adaptável para visualização tanto em computadores quanto em celulares.

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estrutura semântica e divisão de seções da página.
* **CSS3:** Estilização personalizada, uso de variáveis de cores, efeitos de transição (hover) nos cards e layout com Flexbox/Grid.
